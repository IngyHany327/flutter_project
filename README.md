# Kabar - News App

A Flutter news app 

## Screens

- Onboarding flow (logo, 3 pages, country selection, topics, news sources, profile)
- Authentication (login, signup, forgot password, OTP, reset password)
- Main app (homepage, trending, notifications, search, explore, bookmark)
- Content (article detail, comments, author profiles)

## Getting Started

```bash
flutter pub get
flutter run
```

The app starts with a demo navigation screen to browse all screens.

lib/
